# Fateh SEO Keyword Map

Google Sheet:
https://docs.google.com/spreadsheets/d/1wWmQTcLoIIInkbqJlJ1AB5UeH7KOErM_tfop0oCYyoM/edit

Use this sheet as the project keyword and page roadmap.

## Update Rules

- Give every page one primary keyword only.
- Do not reuse the same primary keyword on multiple live pages.
- Keep plumbing city pages and electrical city pages separate.
- Update the sheet whenever a URL, primary keyword, secondary keyword, page status, or priority changes.
- Before creating a new page, check the `Keyword Map` and `City Expansion` tabs first.
- Blog topics should support a service page and link users toward the correct service.
- For pages with a Google review carousel, include at least 7 review cards so desktop shows about 5 review dots and the slider feels active.
- Use blue icon/accent treatment for plumbing pages and red icon/accent treatment for electrical pages, including service icons, problem cards, CTA bands, and process step numbers.
- For future electrical pages, reuse the locked red classes in `public/styles.css`: `electrical-hub-band`, `electrical-trust-banner`, `electrical-problem-band`, `electrical-cta-band`, and `electrical-process-band`.

## Tabs

- `Summary`: quick overview and next action.
- `Keyword Map`: live and planned URLs with primary and secondary keywords.
- `City Expansion`: future city-by-city plumbing and electrical page plan.
- `Content Backlog`: page and blog ideas to build later.
- `Rules`: keyword targeting rules for future work.
- `SERP Validation`: live search/competitor validation notes for the main keyword targets.
